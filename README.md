# She Can Foundation - Frontend Internship Task

## Overview

This project is a simple and responsive single-page website created for the Frontend Development Internship Task at She Can Foundation.

The website is designed to spread awareness about the foundation's mission of empowering women through education, skill development, leadership opportunities, and community support.

## Features

* Responsive Design
* Modern User Interface
* Hero Section
* About Us Section
* Mission Cards
* Impact Statistics
* Volunteer Section
* Interactive Buttons
* Footer Section

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Purpose

The purpose of this project is to demonstrate basic frontend development skills, including webpage structure, styling, responsiveness, and user interaction.
